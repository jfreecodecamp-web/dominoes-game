Dominoes Game (JavaScript)

## Description
An interactive browser-based Dominoes game developed using JavaScript. This project implements core dominoes gameplay including tile matching, turn handling, and user interaction.

## Features
- Tile matching logic
- Turn-based gameplay
- Interactive user interface
- Basic game rules implemented

## Tech Used
- JavaScript
- HTML
- CSS

## Live Demo
https://jfreecodecamp-web.github.io/dominoes-game/

## How to Run
1. Download or clone the repository
2. Open index.html in your browser
